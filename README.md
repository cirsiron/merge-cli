## 合并分支快捷命令

### （install）安装
```js
npm i -g vscode-merge-cli
```

### (usage)使用方法
```javascript
 1. cd到项目的根目录
 2. 全局安装 npm i -g vscode-merge-cli
 3. 执行 vscode-merge-cli, 自动生成vscode task任务文件
 4. 打开vscode快捷键页面，搜索tasks 找到任务运行选项，进行快捷键设置
 5. 执行上面设置的快捷键，根据弹出的框选择或者输入对应分支进行合并
```
